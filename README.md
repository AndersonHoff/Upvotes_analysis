# Upvotes analysis
Analysis and prediction of upvotes from best authors answers.

Working with RMarkdown and testing different machine learning algorithms for prediction.  
Some models include:  
Linear Regresion  
Nonlinear Regresion  
Neural Net  
XGBoost   
Keras
